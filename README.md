# 쿠팡 검색 캡처 (크롬 익스텐션)

쿠팡에서 키워드를 검색하고, 검색 결과 화면을 캡처한 뒤, 화면에 나오는 모든 상품의
**상품명 · 가격 · 별점 · 리뷰수**를 추출해 **PNG + CSV**로 저장하는 Manifest V3 크롬 익스텐션입니다.

## 설치

1. 크롬 주소창에 `chrome://extensions` 입력 → 우측 상단 **개발자 모드** 켜기
2. **압축해제된 확장 프로그램 로드** 클릭 → 이 폴더(`chrome-scraping`) 선택
3. 익스텐션 목록에 "쿠팡 검색 캡처"가 추가됨

## 시크릿 모드에서 사용 (중요)

시크릿 모드 동작은 매니페스트로 강제할 수 없고, 사용자가 직접 허용해야 합니다.

1. `chrome://extensions` → "쿠팡 검색 캡처" 카드의 **세부정보** 클릭
2. **시크릿 모드에서 허용** 토글을 **켜기**
3. 시크릿 창(`Cmd+Shift+N`)을 열고 사용

## 사용법

1. 툴바의 익스텐션 아이콘 클릭
2. 검색 키워드 입력(예: `무선마우스`) → **실행**
3. 새 탭에서 쿠팡 검색 결과가 열리고, 자동으로:
   - 1페이지 전체 상품을 로드(점진적 스크롤)
   - 맨 위로 복귀 후 **화면 캡처(PNG)**
   - 상품 데이터를 **CSV**로 저장
4. 다운로드 폴더에 `coupang_<키워드>_<날짜시간>.png` / `.csv` 두 파일이 생성됩니다.

CSV 컬럼: `상품명, 가격, 별점, 리뷰수, 상품링크` (UTF-8 BOM 포함 → 엑셀에서 한글 정상 표시)

## 동작 구조

| 파일 | 역할 |
|------|------|
| `manifest.json` | MV3 매니페스트 (권한, incognito 설정) |
| `popup.html` / `popup.js` | 키워드 입력 + 실행 UI, 진행 상태 표시 |
| `background.js` | 탭 생성 · 로드 대기 · content 주입 · 화면 캡처 · CSV/PNG 다운로드 |
| `content.js` | 결과 페이지 스크롤 + 상품 정보 추출 |

## 알려진 제약 / 문제 해결

- **상품이 0건으로 나올 때**: 쿠팡이 마크업을 바꿨거나 접속이 차단된 경우입니다.
  검색 탭에서 `F12`(개발자도구)로 상품 요소의 클래스명을 확인한 뒤
  `content.js`의 셀렉터(`getItems`, `firstText`, `getRating`, `getReviewCount`)를 보정하세요.
- **봇 탐지**: 쿠팡은 자동화를 탐지/차단할 수 있습니다. 너무 잦은 반복 실행은 피하세요.
  실제 브라우저에서 동작하므로 헤드리스보다는 안전하지만, 차단 우회는 이 도구의 범위 밖입니다.
- **스크린샷이 비어 있을 때**: 캡처 권한 이슈일 수 있습니다. 익스텐션을 새로고침
  (`chrome://extensions`에서 새로고침 아이콘) 후 다시 시도하세요.
- 캡처는 **전체 페이지(스크롤 바닥까지)** 입니다. 기본적으로 CDP(`Page.captureScreenshot`)로
  **한 번에 단일 이미지**를 찍습니다. 이때 상단에 "…디버깅을 시작했습니다" 배너가 잠깐 뜹니다(정상).
  디버거 사용이 불가하면 자동으로 **뷰포트 여러 장 → 이어붙이기**로 폴백합니다(고정 헤더가 반복될 수 있음).

## 확장 아이디어 (미구현)

- 여러 페이지(페이지네이션) 수집
- 검색창에 실제 타이핑하는 방식
